AI бот Телеграм
Кандидатам на работу в Латокен нужен AI бот, чтобы узнавать у него детали работы в Латокен и процесса интервью.
Бота можно сделать на любом языке программирования или в конструкторе No Code (подходит для Операционных менеджеров без опыта разработки).
Уровни сложности:
No code + О Латокен и Хакатоне в промпте бота: бот отвечает на вопросы о Латокен и Хакатоне.
Code + О Латокен и Хакатоне в промпте бота: как 1, но с помощью кода на любом языке.
Code + О Латокен и Хакатоне + RAG Culture Deck: как 2, плюс Culture Deck в RAG (желательно от openai assistant)
Code + О Латокен и Хакатоне + RAG Culture Deck + Test: Как 3, плюс Тест - бот, ответив на вопрос задает вопрос, тестируя кандидата на понимание этих 3х материалов.
